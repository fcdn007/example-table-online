# example-table-online
线上表格Chat的示例代码

## 示例

1. 安装Node环境
2. 安装supervisor，安装方法：`npm i -g supervisor`
3. 克隆代码到本地，顺便点个Star
4. 进入项目目录，安装依赖，`npm i`
5. 运行项目，`npm start`
